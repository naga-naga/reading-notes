# CLAUDE.md — AWSではじめるインフラ構築入門 第2版

## このディレクトリの方針

本書はAWSコンソールでぽちぽちリソースを構築していく内容だが、以下の理由からすべてTerraformで実装する。

- Terraformの学習を兼ねる
- 読み終えたあとのリソース削除が `terraform destroy` 一発で済む

## ディレクトリ構成

```
codes/   # Terraformコード（本書全体でひとつのルートモジュール）
notes/   # 読書メモ
```

章をまたいでひとつのインフラを段階的に構築していく構成のため、章ごとにディレクトリは分けない。

## Terraformコードの書き方

- `codes/` 配下にひとつの `terraform` ルートモジュールを置く
- リソース名は本書のコンソール手順に倣いつつ、Terraform的な命名に整える
- `terraform.tfvars` や `variables.tf` を活用して設定値を外出しにする
- コンソール手順との対応がわかるようにコメントを残す

## 注意事項

- AWSリソースは課金が発生するため、使い終わったら必ず `terraform destroy` を実行する
- 本書で使用するリージョンは基本的に `ap-northeast-1`（東京）
